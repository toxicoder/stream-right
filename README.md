# Easy Game Streaming with Moonlight and Sunshine

This project provides tools to simplify game streaming setup using Moonlight and Sunshine, specifically optimized for Windows 11 laptops with hybrid graphics.

## Features

- **Virtual Display Management**: Automatically creates and manages virtual displays.
- **Resolution Matching**: Adjusts virtual display resolution to match the client.
- **Power Management**: Turns off physical displays during streaming to save power.
- **GPU Optimization**: Forces dGPU usage on hybrid systems.
